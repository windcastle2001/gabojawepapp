## [2026-05-01] 친구 그룹 모드 — DB 설계 방향

**결정 1:** `couples` 테이블 유지 + 신규 `groups`/`group_members` 테이블 병렬 추가

**이유:**
- `wishlist.couple_id`가 `couples.id`를 FK로 참조 중 — 테이블 교체 시 마이그레이션 비용 과대
- 기존 커플 기능 무중단 보장하면서 신규 그룹 기능 추가

**결정 2:** 멤버 수 제한(`couple=2`, `friends=10`)을 트리거(`fn_check_group_member_limit`)로 강제

**이유:**
- PostgreSQL CHECK 제약은 동일 행 내 컬럼만 참조 가능
- `max_members`가 `groups` 테이블에 있고, 제약 대상은 `group_members` 테이블 → 크로스 테이블 CHECK 불가

**결정 3:** `community_reviews` 신규 테이블로 생성 (`couple_id` 대신 `group_id` FK)

**이유:**
- 리뷰는 그룹 소속 없이 개인이 작성 가능 (group_id NULL 허용)
- `review_type` 컬럼으로 맵 필터링 지원

**결정 4:** `group_members.role` UPDATE는 RLS 정책에서 차단, service_role API에서 처리

**이유:** 일반 유저가 자신을 owner로 승격하는 권한 상승 공격 차단

**영향:**
- back-expert: 그룹 초대 수락, 멤버 역할 변경 API → supabaseAdmin(service_role) 사용 필요
- front-coder: review_type 필터 파라미터 맵 API에 추가 필요

**검증:** [오하나 수석] 2026-05-01 — 스키마 반영 완료

## [2026-05-01] DateMate 디자인 토큰 전환 승인

**결정:** 메인 컬러를 기존 로즈 계열에서 Terracotta `12 72% 52%`로 이동하고, 배경은 크림화이트 `30 31% 96%` 계열로 전환한다. 하단 탭바는 기존 4탭 구조를 유지한다.

**이유:**
- 기존 로즈-핑크 중심 팔레트는 여성 타겟 편향이 강해 커플 양쪽이 함께 쓰는 제품 톤과 충돌
- Terracotta는 따뜻하지만 덜 장식적이고, 저장/계획 도구로서의 신뢰감을 더 잘 제공
- 크림화이트 배경은 순백보다 차갑지 않고 오래 머무르는 앱 톤에 적합
- 4탭(홈/위시/지도/AI)은 PRD의 핵심 흐름과 일치

**반영:**
- `app/globals.css`, `tailwind.config.ts`에 시맨틱 토큰 추가
- `components/ui` 공용 컴포넌트 6종 추가
- 홈/레이아웃/캡처/위시 카드 일부 토큰 기반 전환

**검증:** [박디자인 과장 역할] 2026-05-01 — 토큰 전환 방향 PASS, 잔여 하드코딩 페이지 후속 수정 필요

## [2026-05-01] 커플·친구 모드와 커뮤니티 맵 통합

**결정:** 커플용 DateMate와 친구 약속용 흐름을 한 제품 안에서 모드 선택으로 제공하고, 커뮤니티 맵은 분리하지 않고 통합한다.

**이유:**
- 맵을 분리하면 커플 맵과 친구 맵 모두 초기 데이터가 빈약해짐
- 통합 맵은 리뷰 축적 속도가 빠르고 신규 사용자가 처음 봐도 풍성한 경험을 제공
- 커플/친구 구분은 `💑 커플`, `👥 친구` 리뷰 뱃지와 필터 칩으로 충분히 전달 가능
- 이후 데이터가 쌓이면 `커플 인기 장소 TOP 10`, `친구 모임 인기 장소 TOP 10` 같은 큐레이션으로 확장 가능

**반영:**
- 로그인 전 화면에서 `Google로 시작하기`와 `로그인 없이 둘러보기`를 분리
- 이후 `커플로 사용` / `친구들과 사용`을 선택하도록 구현
- 커플은 파트너 수락 전 초대 상태, 수락 후 초대 버튼 제거 및 연결 해제 상태로 전환
- 친구 모임은 최대 10명까지 초대 카운트를 표시
- 위시리스트 완료 후 리뷰를 커뮤니티 맵에 공유하면 현재 모드에 따라 커플/친구 리뷰로 등록

**검증:** [Directorcode] 2026-05-01 — `npx tsc --noEmit --incremental false` PASS
